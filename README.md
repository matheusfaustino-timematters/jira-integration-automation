# Jira Integration

It is an attempt to **automate** some basic tasks from our ticket system. This would give us more time and take out pretty simple non-brainer tasks, that requires certain steps, file manipulation and triggers to work.


## Run it

- Required python version can be found in file [.python-version](.python-version)
- Requires uv to run it and projected self-installed in the environment:
-- `uv pip install -e .`
-- `uv run src/main.py`
